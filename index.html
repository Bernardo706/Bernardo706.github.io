    • Git é o sistema de controle de versão open source mais usado no mundo atualmente!
    Ele é usado para controlar o histórico de alterações de arquivos e principalmente de projetos
    de desenvolvimento de software.
    Ele permite mais flexibilidade no fluxo de trabalho, segurança e desempenho.



    O GitHub é considerado é uma ferramenta essencial para engenheiros de software, com uma popularidade sem igual.
    Atualmente, ele acomoda mais de 25 milhões de usuários.
    Isso significa que há um número considerável de profissionais que estão procurando o GitHub para melhorar o fluxo de trabalho e a colaboração.

    Em suma, o GitHub é um serviço baseado em nuvem que hospeda um sistema de controle de versão (VCS) chamado Git.
    Ele permite que os desenvolvedores colaborem e façam mudanças em projetos compartilhados enquanto mantêm um registro detalhado do seu progresso.

    Visualiza os arquivos que foram alterados

    git status

    Prepara um arquivo para ser versionado

    git add nome_arquivo

    Gera a versão das modificações

    git commit -m "mensagem"

    Envia para o GitHub

    git push origin main
